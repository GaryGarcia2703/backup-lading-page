# Backup Empresarial – Projeto Front-end

Projeto front-end desenvolvido com foco em UI/UX, responsividade e interação, simulando uma landing page profissional para apresentação de planos de backup empresarial.

Este projeto faz parte do meu portfólio pessoal e representa minha evolução no desenvolvimento web, aplicando boas práticas de layout, organização de código e experiência do usuário.

---

## Objetivo do Projeto

Criar uma interface moderna e clara para apresentar planos de backup, destacando benefícios, diferenciais e chamadas para ação de forma visualmente atraente e intuitiva.

---

## Funcionalidades

- Cards de planos (Básico, Profissional e Avançado)
- Plano destacado com identidade visual própria
- Modal interativo com informações detalhadas de cada plano
- Ícones para diferenciar vantagens e diferenciais
- Animações suaves ao rolar a página
- Layout totalmente responsivo

---

## Tecnologias Utilizadas

- HTML5  
- CSS3  
  - Flexbox  
  - Gradientes  
  - Media Queries  
  - Animações e transições  
- JavaScript (Vanilla)  
  - Manipulação do DOM  
  - Renderização dinâmica de conteúdo  
- Font Awesome  
- AOS.js  

---

## Design e Experiência

- Inspiração visual em Frutiger Aero  
- Paleta de cores suaves com foco em tons de azul  
- Hierarquia tipográfica clara  
- Destaque visual para elementos importantes  
- Interface pensada para legibilidade e fluidez  

---

## Responsividade

O projeto foi desenvolvido para funcionar corretamente em:

- Smartphones  
- Tablets  
- Laptops  
- Telas grandes  

Utilizando media queries e layout flexível.
